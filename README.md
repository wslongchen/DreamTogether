####项目简介

+ 每个人都有自己的梦想，或大或小。
+ 这是一个情怀app。
+ 一群追梦文青的自留地。

**********************************************************

####项目介绍
##### PHP服务端
+ 采用mvc模式设计后段接口,由于暂时只提供接口供安卓端访问,暂时缺少v层。
+ 数据库设计：梦想和用户两张表，以及两张扩展表。自由度增高，方便以后增加信息。
+ 

##### ANDROID应用端
+ 完成了首页显示所有梦想、用户的登录(注册)、发表梦想(暂时只支持文字)、显示用户发表梦想的时间线。
+ 未完待续
